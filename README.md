# Modern Animated Portfolio

A responsive single-page portfolio built with Vite, React, Framer Motion, and Three.js.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize

- Update the hero, about, skills, projects, and contact copy in the component files under src/components.
- Replace the placeholder project cards in src/components/Projects.jsx with your real projects.
- Swap the name, links, and contact details as needed.
